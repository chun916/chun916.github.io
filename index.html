数据结构笔记（8）：树&二叉树基础

什么是树？树有哪些表示形式？

树是n(n>=0)个节点的有限集，当n=0,为空树。

  • 有且只有一个特定的被称为根的结点。
  • 当n>1时，其余节点可分为m(m>0)个互不相交的有限集，其中每个集合本身又是一棵树，称为根的子树。
    树是递归的。

如何熟练地解释结点、结点的度、树的度、叶子、分支结点、孩子、双亲、兄弟、祖先、深度等术语的含义？

结点的度:树中这个结点的孩子个数。
树的度：树中结点最大的度。
叶子结点：度为0，即没有子女结点的结点，又称终端结点
分支结点（非终端节点）：度大于0的结点。

有序树与无序树的区别是什么？

树中结点的各子树从左到右是有次序的，不能互换，称为有序树，否则为无序树。

什么是二叉树？

二叉树或为空树，或是由一个根结点加上两棵分别称为左子树和右子树的、互不相交的二叉树组成。

二叉树有哪几种基本形态？

空树
只含根节点
右子树为空树
左子树为空树
左右子树均不为空树

具有5个结点的二叉树可能有几种不同形态？普通有序树呢？

5个结点的二叉树有42种形态。
n个结点的二叉树形态数：C(n/2n) / (n+1) ,卡特兰数

二叉树有哪两种存储结构？各自是怎样的？

顺序存储和链式存储

适用于满二叉树、完全二叉树
下表为i的结点，左孩子下标：2i+1，右孩子下标：2i+2，双亲下标：(i+1)/2-1

在n个结点的二叉链表中，有n-1个空指针域

二叉链表的结点结构是怎样的？

包含一个数据域和两个指针域。

什么是广义表？广义表的深度、长度各自指什么？是否会求广义表的表头、表尾？

  • 上面的函数是常量成员函数，即，该函数不改变类中成员数据的内容，其中const的位置是在参数列表后，即在()后面，{}前面。在确定函数不改变数据内容的时候，一定要将函数写成常量成员函数的形式，即一定要加const。因为常量类只能调用常量成员函数，若函数不加const，则常量类调用该函数的时候就会出现错误，因为调用非常量成员函数，会改变类的成员数据，即与常量类自相矛盾。
  • access level（访问级别）
  1. public 供外界访问和使用，如外界使用的函数
  2. private 只在class内可用，比如数据部分
  3. protected 受保护的
  • 构造函数 constructor（ctor）
  1. 函数名一定等于类名
  2. double r = 0, double i = 0 ：默认实参，创建函数时没有传参时调用
  3. 没有返回值类型，因为构造函数用来创建对象
  4. : re (r), im (i) //初始值，初始列，构造函数的独有语法，和大括号内的赋值是不同阶段，构造函数中要尽量使用值的初始化，而不是赋值，这样性能会更高。
    大括号中同样可以赋值,但会放弃初始化的阶段，效率低了些。
    构造函数也可放在类的private区域，此时使用构造函数创建类将是错误的，因为private不能被访问到，即不允许外界通过构造函数创建类，单例模式即是如此。
  5. 不带指针的类多半不用写析构函数
  6. ctor 可以有很多个–overloading(重载)
  7. 防止函数重载调用冲突
  8. 构造函数放在private区，是一种单例模式（Singleton)。
  • const
    class类内的函数有两种：会/不会改变数据内容，不会改变数据内容的函数要在()后{}前加上const。
  • 参数的传递：pass by value vs pass by reference(to const) (按值传递或引用传递)
  1. 尽量不用按值传递，尤其是数据占用字节数较多时，值传递会将参数拷贝并传递，最好所有的参数都使用引用传递，引用底层就是指针。
  2. 引用传递形参会改变实参，通过const防止修改数值。

  • 返回值传递： return by value vs return by reference(to const)
    返回值的传递尽量使用引用传递

  • 函数内新创建的变量，函数结束后会销毁所占内存空间，不能使用引用传递。

  • friend（友元）
    通过友元类、友元函数访问类的私有成员变量。

什么是队列？队列的特点是什么？

队列是一种操作受限的线性表，只允许在表的一端进行插入，而在表的另一端进行删除。
队列的特点是先进先出。

什么是循环队列？为什么要构成循环队列？

顺序存储结构的队列会出现 假溢出 的情况，通过循环队列可以解决顺序队列的缺点。
循环队列是将顺序队列臆造为一个环状的空间，即把存储队列元素的表从逻辑上视作一个环。
当首指针Q.front=MaxSize-1后，再前进一个位置就自动到0，用除法取余（%）运算实现。

如何判断循环队列的队空、队满？

front==rear 无法区分队满队空
队满：牺牲一个元素空间，当(rear+1)%MaxQSize==front时，队满
对空：front==rear

循环队列中的元素个数如何得到？

m:队列的容量
元素个数=(rear-front+m) % m
