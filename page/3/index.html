栈与队列 进制转换

Problem

将十进制整数n转换成k进制数。

Input

首先输入一个整数T，表示测试数据的组数，然后是T组测试数据。每组测试数据输入两个整数n和k（ -1000000 ≤ n ≤ 1000000, 2 ≤ k ≤ 9）。

Output

对于每组测试，先输出n, 然后输出一个空格，最后输出对应的k进制数。

  • 注：考虑n等于正、负、零值的情况。

什么是队列？队列的特点是什么？

队列是一种操作受限的线性表，只允许在表的一端进行插入，而在表的另一端进行删除。
队列的特点是先进先出。

什么是循环队列？为什么要构成循环队列？

顺序存储结构的队列会出现 假溢出 的情况，通过循环队列可以解决顺序队列的缺点。
循环队列是将顺序队列臆造为一个环状的空间，即把存储队列元素的表从逻辑上视作一个环。
当首指针Q.front=MaxSize-1后，再前进一个位置就自动到0，用除法取余（%）运算实现。

如何判断循环队列的队空、队满？

队满：牺牲一个元素空间，当(rear+1)%MaxQSize==front时，队满
对空：front==rear

循环队列中的元素个数如何得到？

m:队列的容量
元素个数=(rear-front+m) % m

什么是栈?栈的特点是什么？

栈和队列是限定插入和删除只能在表的“端点”进行的线性表。
栈是限定插入和删除只能在表尾进行的线性表。

特点：

  • 插入位置：表尾，insert(n+1,x)
  • 删除位置：表尾，dalete(n)
  • 后进先出

什么是顺序栈？如何实现顺序栈？

顺序栈是用顺序存储结构表示的栈，用一个顺序表和一个栈顶指针实现。

什么是链栈？如何实现链栈？

先进后出，类似于建立逆序链表。

数据结构笔记（1）：基本概念和算法复杂度

Posted on 2021-01-26 Edited on 2021-05-06 In 数据结构笔记

数据结构基本概念

数据元素 数据项 数据对象

  • 数据元素是数据的基本单位，用于完整的描述一个对象，如一名学生记录。
  • 数据项是组成数据元素的、有独立含义的、不可分割的最小单位，如学生基本信息表的学号、姓名。
  • 数据对象是性质相同的数据元素的集合，是数据的一个子集。

数据结构

  • 什么是数据结构？
  • 数据结构是相互之间存在一种或多种特定关系的数据元素的集合，结构就是指数据元素之间的关系。
  • 数据结构学习的三个层次？
  1. 数据结构的基本知识和概念
  2. 数据结构的应用
  3. 算法的实现

  • 数据结构包括逻辑结构和存储结构，是问题的数据模型

    数据结构的三要素s

  • 数据的逻辑元素

  • 数据的存储结构

  • 数据的运算

    逻辑结构

    数据的逻辑结构从逻辑上描述数据，与数据的存储无关。

    存储结构

    存储结构又称物理结构，把数据对象存储到计算机时，既要求存储数据元素的数据，又要存储数据元素之间的逻辑关系。
    存储结构包括顺序存储结构和链式存储结构。

  • 顺序存储结构：借助元素在存储器的相对位置表示数据元素之间的逻辑关系，所有元素依次存放在一片连续的存储空间内。

  • 链式存储通过给每个节点附加指向后继元素地址的指针表示节点间的关系。

    算法和算法分析

    算法的定义

    算法是为了解决某类问题而规定的一个有限长的操作序列。

    算法的特性

  • 有穷性：执行有穷步且在又穷时间内完成。

  • 确定性：每种情况所执行的操作，在算法中都有确定的规定，不会产生二义性。

  • 可行性

  • 输入： 有零个或多个输入

  • 输出： 有一个或多个输出

    算法的时间复杂度

  • 影响算法时间代价的最主要因素是问题规模，问题规模是算法求解问题输入量的多少，是问题大小的本质表示。

  • 一条语句的重复执行次数称作语句频度，一个算法的执行时间可用该算法中所有语句频度之和来度量。

  • 常量阶时间复杂度：与问题的规模无关，执行时间恒定的算法。
    for(i=0;i<1000;i++){x++;s=0;}
    算法的执行时间为常数，即使常数再大，时间复杂度都是O(1)。

    算法的空间复杂度

    空间复杂的度量算法的存储空间，也是问题规模n的函数，一般只需分析算法在实现时所需的辅助空间。

  • 算法原地工作：算法执行所需辅助空间相对输入量而言是常数，空间复杂度为O（1）。

    只有一个辅助变量t，与问题规模n无关，空间复杂度为O（1）。
